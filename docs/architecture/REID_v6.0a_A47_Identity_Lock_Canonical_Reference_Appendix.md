## 1. Purpose
The Identity-Lock Canonical Reference Appendix (ILCRA) consolidates all identity-lock rules,
invariants, primitives, continuity rules, drift-immunity rules, fusion rules, synthesis rules,
governance rules, and lineage rules into a single canonical reference.

ILCRA ensures:
- unified identity-lock understanding
- unified continuity understanding
- unified drift-immunity understanding
- unified fusion/synthesis understanding
- unified governance understanding
- unified lineage understanding

ILCRA is mandatory for all v6.0a surfaces and workflows.
