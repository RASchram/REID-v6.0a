## 1. Purpose
The Identity-Lock Governance Standard (ILGS) defines the supervisory governance layer that ensures:
- identity-lock correctness
- continuity correctness
- drift-immunity correctness
- fusion correctness
- synthesis correctness
- lineage correctness

ILGS ensures:
- safe identity evolution
- deterministic identity behavior
- predictable multi-surface identity
- unified governance across all identity operations

ILGS is mandatory for all v6.0a surfaces and workflows.
