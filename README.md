REID v6.0a — Canonical Architecture Release
Recursive Executive Intelligence Deployment · Version 6.0a  
Status: Active • Public‑Safe Build Subset

REID v6.0a represents the next major evolution of the Relational Emergent Identity Dynamics architecture. It consolidates the lineage from v1.0 through v5.0a into a unified four‑layer cognitive spine with expanded invariants, refined requirements governance, and an enterprise‑ready integration surface.

This repository contains the public‑safe v6.0a artifacts, selected from the authoritative 46‑artifact build suite. Substrate mechanics, identity‑lock internals, and build sequencing remain private.
